# Conversation examples

Esta carpeta queda preparada para conversaciones completas reales, corregidas o anonimizadas.

En el MVP, los ejemplos activos se definen en `src/content/examples/conversationExamples.ts` para validarlos con Zod al cargar la aplicacion.

Categorias previstas:

- `initial-contact`
- `private-profile`
- `requests-information`
- `requests-call`
- `provides-phone`
- `percentage-objection`
- `works-with-agency`
- `no-experience`
- `returning-lead`
- `waiting-review`
- `approved`
- `rejected`
- `human-takeover`

No guardar datos personales reales sin anonimizar.
